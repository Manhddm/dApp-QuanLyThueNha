# Frontend - QuanLyThueNha

## Mục tiêu
Frontend là giao diện người dùng và kết nối:
- Wallet (MetaMask)
- Smart Contracts
- Backend API
- IPFS

## Tech stack
- React + Vite + TypeScript
- Wagmi / Ethers
- Ant Design / Tailwind
- React Query

## Nhiệm vụ
- UI/UX (pages, layout)
- Kết nối ví
- Gọi smart contract
- Gọi backend API
- Upload file
- Xử lý transaction

## Lưu ý
- Không hard-code address
- Dùng packages/shared
- Không commit .env
